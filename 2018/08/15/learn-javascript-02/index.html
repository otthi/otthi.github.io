「ノンプログラマのためのJavaScript」 第2章 JavaScriptの文法 より

  • 変数とは、データを保存しておく箱
  • どんなデータでも入れることができる

試してみたこと

変数にはどんなデータでも入れることができます。
P.24より

入れる値によってどんなデータ型が返ってくるか、typeof を使って確認してみる。

var a;
typeof a //-> "undefined"
a = 1;
typeof a //-> "number"
a = '1';
typeof a //-> "string"
a = [1, 2, 3];
typeof a //-> "object"
a = {b: 1}
typeof a //-> "object"

配列もオブジェクトも object らしい。
データの種類については、本を読み進めていく方が理解できそうだったので、今はこれを疑問に持ちながら先に進んでいこうと思う。
